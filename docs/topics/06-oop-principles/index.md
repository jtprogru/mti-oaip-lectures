# Тема 6. Основные принципы объектно-ориентированного программирования (OOP Principles)

Тема знакомит с базовыми понятиями ООП (класс, объект, интерфейс), тремя ключевыми принципами (инкапсуляция, наследование, полиморфизм) и сравнивает реализацию ООП в Python и Go. Вторая лекция расширяет тему — событийно-управляемая модель программирования и компонентно-ориентированный подход.

## Лекции

- [Лекция 1. Базовые понятия ООП. Инкапсуляция, наследование, полиморфизм](lecture-01-oop-basics.md) — история, классы/объекты/интерфейсы, три принципа, реализация в Python и Go.
- [Лекция 2. Событийно-управляемая модель и компонентно-ориентированный подход](lecture-02-event-driven.md) — генераторы, корутины, asyncio в Python, горутины в Go, КОП.
