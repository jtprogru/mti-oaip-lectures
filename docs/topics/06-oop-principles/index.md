# Тема 6. Основные принципы объектно-ориентированного программирования (OOP Principles)

Тема знакомит с базовыми понятиями ООП (класс, объект, интерфейс), тремя ключевыми принципами (инкапсуляция, наследование, полиморфизм) и сравнивает реализацию ООП в Python и Go. Вторая лекция расширяет тему до событийно-управляемой модели программирования и компонентно-ориентированного подхода. Третья — про SOLID-принципы как мост к шаблонам проектирования.

## Лекции

- [Лекция 1. Базовые понятия ООП. Инкапсуляция, наследование, полиморфизм](lecture-01-oop-basics.md) — история, классы/объекты/интерфейсы, три принципа, реализация в Python и Go.
- [Лекция 2. Событийно-управляемая модель и компонентно-ориентированный подход](lecture-02-event-driven.md) — генераторы, корутины, asyncio в Python, горутины в Go, КОП.
- [Лекция 3. SOLID-принципы](lecture-03-solid.md) — SRP, OCP, LSP, ISP, DIP с антипримерами и рефакторингом на Python и Go.
