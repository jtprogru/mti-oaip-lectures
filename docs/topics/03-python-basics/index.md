# Тема 3. Программирование на Python (Python Basics)

!!! info "В разработке"
    Содержимое темы переносится из [`articles/`](https://github.com/jtprogru/mti-oaip-lectures/tree/main/articles). Pascal-фрагменты исходных лекций удаляются — остаётся Python (плюс псевдокод для алгоритмических разделов).

## Лекции

- **Лекция 1.** Основные элементы языка. Структурная схема программы. Лексика. Переменные и константы. Типы данных. Выражения и операции.
- **Лекция 2.** Операторы языка. Синтаксис, составной оператор, вложенные условные операторы. Циклические конструкции.
- **Лекция 3.** Ввод/вывод в консольных приложениях. Массивы как структурированный тип данных. Одномерные и двумерные массивы.
- **Лекция 4.** Строки и множества. Объявление, операции. Стандартные функции и процедуры работы со строками.
- **Лекция 5.** Кодировки ASCII, UNICODE. Списки, кортежи, словари. Множества и операции над ними.

## Лабораторные работы

- Работа в среде программирования. Реализация построенных алгоритмов.
- Программы линейной и разветвляющейся структуры.
- Программы циклической структуры. Обработка одномерных и двумерных массивов.
- Работа со строковыми переменными. Работа с данными типа множество.
