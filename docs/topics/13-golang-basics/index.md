# Тема 13. Основы Go (Golang Basics)

!!! info "В разработке"
    Тема добавляется в Фазе 4 — после стабилизации Python-материала.

## Планируемые лекции

- **Лекция 1.** Знакомство с Go: установка, `go mod`, структура проекта, первая программа.
- **Лекция 2.** Переменные, типы, константы. Управляющие конструкции (`if`, `for`, `switch`).
- **Лекция 3.** Функции, множественный возврат, замыкания, `defer`. Обработка ошибок.
- **Лекция 4.** Композитные типы: массивы, срезы, карты, структуры.
- **Лекция 5.** Методы и интерфейсы. Полиморфизм через интерфейсы.
- **Лекция 6.** Пакеты, модули, импорт. Тестирование: `testing`, табличные тесты.
