# Тема 11. Разработка приложений (Application Development)

Тема охватывает практические аспекты разработки приложений: логирование, веб-разработка на Django, сборка в самодостаточные исполняемые файлы.

## Лекции

- [Лекция 1. Логирование](lecture-01-logging.md) — модуль `logging` в Python, конфигурация через словарь, ротация логов, JSON-формат, `log/slog` в Go.
- [Лекция 2. Django — основы фреймворка](lecture-02-django-basics.md) — MVT, модели и миграции, ORM, маршруты, views, шаблоны, готовая админка.
- [Лекция 3. Django — формы, сигналы и REST API](lecture-03-django-events-api.md) — `ModelForm`, аутентификация, сигналы, Django REST framework, JWT.
- [Лекция 4. Компиляция в standalone](lecture-04-standalone.md) — PyInstaller, Nuitka, Briefcase для Python; `go build` и кросс-компиляция для Go; Docker как альтернатива.

## Лабораторные работы

- Разработка оконного приложения.
- Разработка оконного приложения с несколькими формами.
