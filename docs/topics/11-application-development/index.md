# Тема 11. Разработка приложений (Application Development)

Тема охватывает практические аспекты разработки приложений: логирование, веб-разработка (Django), сборка в самодостаточные исполняемые файлы.

## Лекции

- [Лекция 1. Логирование](lecture-01-logging.md) — модуль `logging` в Python, конфигурация через словарь, ротация логов, JSON-формат, `log/slog` в Go.
- **Лекция 2. Django — основы фреймворка.** В разработке — обзор Django как полнофункционального веб-фреймворка: MTV-архитектура, ORM, миграции, админка.
- **Лекция 3. Django — обработка событий и API.** В разработке — формы, аутентификация, REST API через Django REST Framework.
- **Лекция 4. Компиляция в standalone.** В разработке — PyInstaller / Nuitka для Python; `go build` для Go (нативный бинарь из коробки).

## Лабораторные работы

- Разработка оконного приложения.
- Разработка оконного приложения с несколькими формами.
