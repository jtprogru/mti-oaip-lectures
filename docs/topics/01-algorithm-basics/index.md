# Тема 1. Основные принципы алгоритмизации и программирования (Algorithm Basics)

!!! info "В разработке"
    Содержимое темы переносится из [`articles/`](https://github.com/jtprogru/mti-oaip-lectures/tree/main/articles).

## Лекции

- **Лекция 1.** Основные понятия алгоритмизации. Свойства алгоритма. Формы записи. Сложность алгоритмов.
- **Лекция 2.** Алгоритмические конструкции: линейные, разветвляющиеся, циклические. Графическое отображение.
- **Лекции 3–4.** Логические основы. Алгебра логики. Конъюнкция, дизъюнкция, инверсия. Таблицы истинности.

## Лабораторные работы

- Разработка алгоритмов для конкретных задач.
- Использование программ для графического отображения алгоритмов.
- Определение сложности работы алгоритмов.
