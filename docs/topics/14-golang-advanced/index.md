# Тема 14. Продвинутый Go (Golang Advanced)

!!! info "В разработке"
    Тема добавляется в Фазе 4 (опционально) — после освоения основ.

## Планируемые лекции

- **Лекция 1.** Конкурентность: горутины, каналы, `select`. Паттерны.
- **Лекция 2.** Контексты, отмена, таймауты (`context`).
- **Лекция 3.** HTTP-клиент и HTTP-сервер на стандартной библиотеке.
- **Лекция 4.** Работа с файлами, JSON, базами данных (`database/sql`).
- **Лекция 5.** Бенчмарки, профилирование, `pprof`. Race detector.
