# Тема 9. Иерархия классов (Class Hierarchies)

Тема даёт обзор стандартной библиотеки Python (плюс стандартных пакетов Go для сравнения) и разбирает классические шаблоны проектирования с параллельными примерами на двух языках.

## Лекции

- [Лекция 1. Стандартная библиотека Python и стандартные модули Go](lecture-01-stdlib.md) — встроенные функции, модули `sys`, `os`, `datetime`, `collections`, `contextlib`, `re`, `json`, `sqlite3`, `threading`/`asyncio`, `unittest`, `logging`; параллельные модули Go.
- [Лекция 2. Шаблоны проектирования](lecture-02-patterns.md) — порождающие, структурные, поведенческие шаблоны с примерами на Python и Go.

## Лабораторные работы

- Создание объектно-ориентированного приложения по индивидуальным заданиям.
