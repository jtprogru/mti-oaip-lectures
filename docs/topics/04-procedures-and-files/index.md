# Тема 4. Процедуры, функции и работа с файлами (Procedures and Files)

!!! info "В разработке"
    Содержимое темы переносится из [`articles/`](https://github.com/jtprogru/mti-oaip-lectures/tree/main/articles).

## Лекции

- **Лекция 1.** Подпрограммы. Процедуры и функции: сущность, назначение, различие. Стандартные процедуры. Процедуры пользователя: синтаксис, передача аргументов, возврат результата.
- **Лекция 2.** Формальные и фактические параметры. Процедуры с параметрами, описание процедур.
- **Лекция 3.** Функции: организация, описание, вызов. Рекурсия. Рекурсивные алгоритмы.
- **Лекция 4.** Типы файлов. Доступ к файлам. Файлы последовательного и произвольного доступа.
- **Лекция 5.** Запись и считывание из файла произвольного доступа. Стандартные процедуры и функции для файлов.

## Лабораторные работы

- Организация и использование процедур.
- Организация и использование функций.
- Работа с файлами последовательного и произвольного доступа.
