## Лекции

- **Лекция 1.** Языки программирования: эволюция, классификация. Системы программирования. Исходный, объектный, загрузочный модули. Интегрированная среда.
- **Лекция 2.** Методы программирования: структурный, модульный, ООП. Жизненный цикл ПО. Типы приложений. Консольные приложения.
