# Конвенции репозитория

Документ фиксирует соглашения по структуре и именованию, чтобы материалы курса оставались единообразными.

## Структура

```
docs/
├── index.md                       # главная страница сайта (оглавление курса)
├── topics/
│   └── NN-kebab-case-english/
│       ├── index.md               # обзор темы
│       └── lecture-NN-short-slug.md
├── labs/
│   └── lab-NN-short-slug.md
├── practice/
├── tickets/
└── assets/img/topic-NN/...
src/
├── python/                        # примеры на Python
└── golang/                        # примеры на Go (Фаза 4+)
tasks/
└── loops/                         # учебные задачи (англоязычные подкаталоги)
```

## Именование

- **Папки тем:** `NN-kebab-case-english/`, где `NN` — двузначный номер с ведущим нулём (`01-`, `02-`, …, `14-`).
- **Лекции внутри темы:** `lecture-NN-short-slug.md`.
- **Лабораторные:** `lab-NN-short-slug.md`.
- **Изображения:** `docs/assets/img/topic-NN/lecture-NN-pPP.{png,gif}` — где `pPP` — порядковый номер картинки внутри лекции.
- **Подкаталоги задач:** на латинице kebab-case (`loops/`, `arrays/`, `strings/`, …).

## Заголовки в markdown

- **H1 файла темы (`index.md`):** `# Тема N. Русское название (English Name)`.
- **H1 файла лекции:** `# Лекция N. Тема лекции (Lecture Topic)` либо в виде H2 внутри обзорного `index.md` темы.
- В тексте используем русский язык как основной; англоязычные термины — в скобках при первом упоминании или там, где это устоявшаяся форма (`HTTP-клиент`, `Django`).

## Контент

- **Языки программирования:** Python (основной), Go (Фаза 4+). Pascal и Delphi из исходных материалов вычищаются — на их месте либо Python, либо псевдокод (для алгоритмических разделов).
- **Markdown:** не оборачивать абзацы вручную на фиксированную длину. Один абзац — одна длинная строка; перенос делает рендерер. Структурные переносы (между абзацами, списки, таблицы, code fences) — как обычно.
- **Code fences:** обязательно указывать язык (` ```python `, ` ```go `, ` ```bash `).
- **Mermaid:** используем плагин `mermaid2` — диаграммы через ` ```mermaid `.

## Локальная проверка сайта

```bash
uv sync
uv run mkdocs serve          # локальный preview на http://127.0.0.1:8000
uv run mkdocs build --strict # сборка с проверкой битых ссылок
```

## Git

- Сообщения коммитов — на русском, в формате [Conventional Commits](https://www.conventionalcommits.org/) (`feat:`, `fix:`, `docs:`, `chore:`, `build:`, `refactor:` и т. д.).
- Один логический шаг — один коммит. Миграция темы из `articles/` — отдельный коммит/PR.
